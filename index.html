FireUp — Travel Booking App

Year: 2022 • Role: UI/UX Designer • Tools: Figma • Type: College Assignment

FireUp adalah aplikasi mobile untuk booking tiket pesawat, hotel, dan bus dalam satu platform, mirip seperti Traveloka. Fokus pada proses booking yang mudah dan user-friendly.

About the Project

FireUp dirancang sebagai solusi all-in-one untuk kebutuhan perjalanan, menggabungkan booking pesawat, hotel, dan bus dalam satu aplikasi mobile yang modern dan intuitif.

Key Features
  • Flight Booking — Cari dan pesan tiket pesawat dengan filter harga dan jadwal
  • Hotel Booking — Cari hotel dengan review, rating, dan foto lengkap
  • Bus Booking — Pesan tiket bus antar kota dengan pilihan kursi interaktif
  • User Profile — Kelola informasi pribadi dan riwayat pemesanan
  • Payment Integration — Multiple payment methods untuk kemudahan transaksi
Design Process
  1. Research — Riset kompetitor (Traveloka, Tiket.com) dan identifikasi kebutuhan pengguna untuk proses booking yang mudah
  2. Wireframing — Merancang struktur halaman dan user flow untuk setiap fitur booking
  3. UI Design — Membuat desain visual yang modern, clean, dan konsisten menggunakan Figma
  4. Prototyping — Membuat interactive prototype untuk simulasi user journey lengkap
Screens Designed

Splash Screen • Home • Search & Results • Flight Details • Hotel Details • Bus Booking • Seat Selection • Payment • Confirmation • Profile • Order History

Design Highlights
  • ✓ Clean & modern interface dengan visual hierarchy yang jelas
  • ✓ Simple booking flow dengan minimal steps untuk mengurangi friction
  • ✓ Konsisten design system dengan reusable components
  • ✓ Intuitive navigation dengan bottom tab bar
  • ✓ Visual feedback pada setiap interaksi user
Key Learnings
  • Memahami pentingnya user flow dalam design untuk mengurangi drop-off rate
  • Menggunakan design system untuk konsistensi dan efisiensi development
  • Membuat interface yang tidak hanya menarik tapi juga fungsional dan praktis
  • Importance of simplifying complex booking processes

SIANTI — Licensing & Tax Management App

Year: Juni 2023 • Role: UI/UX Designer • Tools: Figma • Type: Internship (Kominfo) • Status: Live on Play Store

SIANTI adalah aplikasi mobile untuk mengelola data perusahaan terkait perizinan dan pajak terhadap SDPPI. Dikerjakan saat magang di Kementerian Komunikasi dan Informatika.

About the Project

Aplikasi ini dikembangkan untuk memudahkan perusahaan dalam mengakses dan mengelola informasi perizinan serta kewajiban pajak yang berkaitan dengan SDPPI (Sumber Daya dan Perangkat Pos dan Informatika). Proyek ini dikerjakan selama masa magang di Kementerian Komunikasi dan Informatika.

Key Features
  • Company Data Management — Kelola data perusahaan secara digital dengan mudah
  • Licensing Information — Akses informasi perizinan SDPPI dengan interface yang jelas
  • Tax Management — Monitoring kewajiban pajak perusahaan secara real-time
  • Document Access — Akses dan download dokumen perizinan dengan mudah
  • Notification System — Alert untuk deadline perizinan dan pajak
Design Process & Challenges
  1. Requirements Gathering — Bekerja sama dengan tim Kominfo untuk memahami kebutuhan user dan sistem perizinan SDPPI yang kompleks
  2. Wireframing & User Flow — Merancang struktur informasi yang kompleks menjadi interface yang mudah dipahami
  3. UI Design — Membuat desain yang profesional dan sesuai dengan guideline aplikasi pemerintah
  4. Design Handoff — Kolaborasi dengan developer untuk implementasi ke aplikasi production
Challenges & Solutions

Challenge: Menampilkan data perizinan dan pajak yang sangat kompleks
Solution: Menggunakan card system dan categorization untuk memudahkan navigasi dan pemahaman informasi

Challenge: Target user dengan tingkat digital literacy berbeda
Solution: Design yang simple dan intuitive dengan clear labeling dan helpful tooltips

Design Highlights
  • ✓ Professional & clean interface untuk aplikasi B2B government
  • ✓ Information architecture yang jelas untuk data kompleks
  • ✓ Accessible design sesuai standar pemerintahan Indonesia
  • ✓ User-friendly meskipun menangani data teknis perizinan
  • ✓ Clear visual hierarchy untuk prioritas informasi
Impact
  • ✓ Aplikasi berhasil diluncurkan di Google Play Store
  • ✓ Digunakan oleh perusahaan untuk mengelola perizinan SDPPI
  • ✓ Membantu digitalisasi proses perizinan di Kominfo
  • ✓ Mengurangi waktu proses pengurusan perizinan
Key Learnings
  • Pengalaman bekerja dalam project pemerintahan dengan requirement spesifik dan formal
  • Kolaborasi dengan stakeholder teknis dan development team
  • Merancang UI/UX untuk aplikasi B2B dengan data kompleks
  • End-to-end process dari design hingga deployment ke production
  • Importance of accessibility dan compliance dalam government apps

CLA Indonesia — Legal Service Website

Year: Juni 2024 • Role: UI/UX Designer • Tools: Figma • Type: Freelance • Status: Live Website

Website corporate untuk firma legal yang menyediakan berbagai paket layanan legal untuk perusahaan. Fokus pada trust, profesionalisme, dan kemudahan akses informasi layanan.

About the Project

CLA Indonesia adalah platform digital untuk perusahaan konsultan hukum yang menyediakan layanan pengurusan legalitas bisnis. Website ini dirancang untuk memudahkan klien dalam memilih paket layanan yang sesuai dengan kebutuhan perusahaan mereka, dengan tampilan profesional yang membangun trust.

Key Features
  • Service Packages — Paket layanan Tax, Legal, dan Associate dengan detail lengkap
  • Service Catalog — Detail informasi setiap layanan yang ditawarkan dengan pricing
  • Company Profile — Informasi lengkap tentang perusahaan dan tim
  • Contact & Consultation — Fitur untuk request konsultasi dan inquiry
  • Responsive Design — Optimal di desktop, tablet, dan mobile
  • Testimonials — Client reviews untuk membangun kredibilitas
Service Categories

Tax Services — Layanan konsultasi dan pengurusan perpajakan perusahaan
Legal Services — Layanan konsultasi hukum dan pengurusan legalitas bisnis
Associate Services — Layanan pendampingan dan partnership untuk kebutuhan legal perusahaan

Design Process
  1. Client Brief & Research — Memahami kebutuhan bisnis klien dan target market (perusahaan yang membutuhkan legal service)
  2. Information Architecture — Merancang struktur konten dan kategori layanan yang mudah dipahami
  3. UI Design — Membuat desain profesional dan trustworthy sesuai dengan industri legal
  4. Responsive Design — Memastikan website optimal di berbagai device dan screen size
Challenges & Solutions

Challenge: Menampilkan berbagai paket layanan yang kompleks
Solution: Menggunakan card-based layout dengan categorization yang jelas dan comparison table

Challenge: Membangun trust untuk layanan legal online
Solution: Design profesional dengan testimonial, credential display, dan informasi perusahaan yang kredibel

Design Highlights
  • ✓ Professional & trustworthy design untuk industri legal
  • ✓ Clear service categorization dengan visual hierarchy
  • ✓ Easy navigation untuk akses informasi paket layanan
  • ✓ Clean layout yang fokus pada konten dan CTA
  • ✓ Responsive design untuk semua device dengan consistent experience
  • ✓ Strategic use of white space untuk professional look
Impact
  • ✓ Website berhasil diluncurkan dan live di production
  • ✓ Meningkatkan digital presence dan kredibilitas perusahaan
  • ✓ Memudahkan calon klien untuk memahami dan memilih layanan
  • ✓ Conversion rate meningkat dengan clear CTA placement
Key Learnings
  • Merancang UI/UX untuk website corporate di industri legal yang memerlukan trust
  • Pengalaman freelance project dari brief hingga deployment
  • Memahami kebutuhan bisnis B2B dalam presentation layanan
  • Responsive web design best practices untuk berbagai device
  • Balance antara aesthetic dan conversion optimization

Universitas Mercu Buana — Website Redesign

Year: Juli 2024 • Role: UI/UX Designer • Tools: Figma • Type: Freelance

Redesign website universitas untuk meningkatkan user experience, modernisasi tampilan, dan kemudahan akses informasi akademik untuk mahasiswa, dosen, dan calon mahasiswa.

About the Project

Project redesign website Universitas Mercu Buana bertujuan untuk menciptakan platform digital yang lebih modern, informatif, dan user-friendly. Website dirancang dengan fokus pada kemudahan akses informasi akademik dan kegiatan kampus untuk berbagai stakeholder.

Key Pages Redesigned (12 Pages)

1. Home Page — Hero section dengan visual kampus menarik, quick access ke informasi penting, section berita dan event kampus, testimonial dan achievement highlights

2. Dosen (Lecturer) — Profile dosen dengan foto profesional, Dean's message section, Sekretaris Prodi information, easy navigation

3. Mahasiswa (Student) — Program studi information, Faculty & Programme study details, Upload area untuk tugas dan dokumen, Student resources

4. Kurikulum — Detailed curriculum information, CPI Program visualization, Scientific vision explanation, Program structure yang mudah dipahami

5. PJM (Quality Assurance) — PJM standard information, Quality assurance guidelines, Upload system untuk dokumen, 360-degree evaluation info

6. Tridharma — Tridharma Perguruan Tinggi information, Upload proposal & laporan system, Academic activity documentation

7. P.Akademik (Academic Guidance) — Pembimbing Akademik system, Upload laporan PA functionality, Form unduh diri (withdrawal form)

8. Kegiatan (Activities) — Campus events & activities, Student engagement programs, Event calendar and registration

9. Kerja Praktek (Internship) — Internship program information, Application and documentation upload, Guidelines for students

10. Library — Library resources access, Podcast section, Digital collections

11. Prestasi (Achievements) — Student & university achievements showcase, Awards and recognition

12. Contact Us — Contact form, Location information, Support channels

Design Features

Modern Visual Design — Color scheme: Dark teal & lime green untuk brand identity, Clean typography untuk readability, Consistent spacing dan layout grid

Information Architecture — Clear navigation struktur, Categorized content untuk easy access, Search-friendly organization

Interactive Elements — Upload document functionality, Download center untuk forms, Event calendar integration, News & announcement section

Responsive Design — Mobile-friendly layout, Tablet optimization, Desktop full experience

User Types Considered
  • Calon Mahasiswa — Informasi program studi dan pendaftaran
  • Mahasiswa Aktif — Akses kurikulum, upload tugas, info event
  • Dosen — Profile management, academic information
  • Stakeholder — Quality assurance, achievement information
Challenges & Solutions

Challenge: Menampilkan banyak informasi akademik yang kompleks
Solution: Information hierarchy yang jelas dengan section-based layout dan progressive disclosure

Challenge: Accommodating berbagai user type dengan kebutuhan berbeda
Solution: Clear navigation dan categorization per user type dengan personalized dashboard

Challenge: Modernisasi tampilan tanpa kehilangan kredibilitas institusi
Solution: Balance antara modern design dengan professional look menggunakan brand colors

Design Highlights
  • ✓ Modern & professional look untuk institusi pendidikan
  • ✓ User-friendly navigation untuk berbagai user type
  • ✓ Comprehensive information architecture dengan 12+ pages
  • ✓ Interactive upload/download features untuk dokumen
  • ✓ Clear visual hierarchy untuk konten akademik
  • ✓ Consistent brand identity throughout all pages
  • ✓ Event calendar dan news system integration
Key Learnings
  • Pengalaman redesign website untuk institusi pendidikan dengan kompleksitas tinggi
  • Memahami kebutuhan multi-stakeholder dalam satu platform
  • Information architecture untuk konten akademik yang kompleks dan beragam
  • Balance antara aesthetic dan functionality untuk educational website
  • Importance of accessibility dan usability dalam education platforms
